# Mina Dotfiles

Detta repository innehåller mina personliga konfigurationsfiler (dotfiles) för att sätta upp min utvecklingsmiljö. Syftet är att samla alla mina inställningar på ett ställe så att jag enkelt kan återställa eller synka min miljö.

## Mappstruktur

Här är en översikt över vad mapparna innehåller:

* **config/**
    Här ligger konfigurationsfiler för olika program.
    
* **scripts/**
    Mina egna Bash-scripts för att automatisera uppgifter.
    
* **zsh/**
    Inställningar specifikt för mitt skal (Zsh), t.ex. alias, prompt och miljövariabler.
